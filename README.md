### 🌩️ Overview

Welcome to my **Google Cloud Cybersecurity Journey** — a hands-on portfolio documenting my progress through the  
🎓 [**Google Cloud Cybersecurity Certificate**](https://www.coursera.org/professional-certificates/google-cloud-cybersecurity) and the complementary [**Security Engineer Learning Path (Path 15)**](https://www.cloudskillsboost.google/paths/15) on **Google Cloud Skills Boost**.

This repository is more than coursework — it’s my growing collection of **projects, labs, notes, and applied security concepts** built entirely in real Google Cloud environments.

### 🧠 Purpose

- 🚀 Build practical, job-ready skills in cloud security  
- 🔐 Demonstrate applied understanding of **Google Cloud Platform (GCP)**  
- 🧩 Document each lab and takeaway for future reference  
- 💼 Showcase my continuous learning journey to potential employers and collaborators

## 🧩 Key Skills & Tools

### 🔐 Cloud Security Concepts
- Shared responsibility, zero trust, and least privilege models  
- Data encryption (KMS, CMEK) and access control enforcement  
- Security posture management using SCC and Cloud Armor  

### 🧰 Technical Tools
- **GCP Core Services:** Compute Engine, VPC, Cloud Armor, SCC, Cloud Logging, IAM  
- **Security Tools:** Security Command Center, Forseti, Cloud Monitoring, Cloud Audit Logs  
- **Languages & Config:** Python, YAML, Bash, Terraform (intro-level)

### 🧠 Soft Skills
- Risk assessment and documentation  
- Security automation mindset  
- Clear reporting and incident summarization
